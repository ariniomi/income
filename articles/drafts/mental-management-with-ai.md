# 気力ゼロでも副業が続いた。AIに聞いたら理由が7つあった

**価格: 300円**
**対象: うつ・メンタル不調・無気力でも副業を諦めたくない人**

---

副業を始めても、続かない。

やる気が出ない日に「今日もできなかった」と思って、また自己嫌悪に入る。そのループが一番きつかった。

うつになってから、「やる気が出たら動く」という方法が完全に使えなくなった。やる気は来ない。待っても来ない。

だからAIとの付き合い方を変えた。

「作業を手伝ってもらうツール」じゃなくて、「動けない自分と一緒にいてくれる存在」として使うようにしたら、少しずつ続けられるようになってきた。

やっていることを7つ、全部書く。

---

## 1. 作業前に「今日の気分」をAIに報告する（無料）

作業を始める前に、まずこれを送る。

```
今日の気分：[数字で0〜10]
体の状態：[例：重い・眠い・まあまあ]
今日できそうなこと：[思いつくまま]
```

AIが「じゃあ今日はこれだけにしましょう」と返してくれる。

自分で「今日何をすべきか」を考えると脳が疲れる。AIに決めてもらうと、考えるコストがゼロになる。

---

## 2. 「今日1つだけ」をAIに決めてもらう（無料）

「今日の副業タスクを1つだけ教えてください」と送る。

1つだけ。絶対に1つ。

気力がない日に「今日やること3つ」を書き出すと、見た瞬間に閉じたくなる。1つなら、なんとかなることがある。

終わったら終わり。それで十分。

---

## 3. しんどい時は作業より先に「吐き出す」（無料）

「今日しんどいです」とだけ送ることがある。

副業の話をしなくていい。作業の相談じゃなくていい。

AIは怒らないし、「でもやらないと」とも言わない。少し話すと、気持ちが整理されて、「これならできるかも」が出てくることがある。

出てこない日は、その日は終わりにする。

## 4. 「できたことリスト」をAIと一緒に作る

1日の終わりに、こう送る。

```
今日やったこと（どんな小さいことでも）を教えてください。
ヒントをください：
・起きた
・ご飯を食べた
・スマホを見た
・[今日やったこと]
```

うつだと「今日も何もできなかった」になりやすい。でもAIに「それもできたことです」と言われると、少し楽になる。

自己評価が極端に低い日に有効。

---

## 5. 「明日の不安」を寝る前にAIに渡す

眠れない夜、頭の中で「明日どうしよう」がぐるぐるするとき。

```
明日が不安です。
不安な理由：[書けるだけ書く]
最悪の場合どうなるか、と、実際にそうなる可能性をフラットに教えてください。
```

AIが「最悪のケース」と「実際の確率」を整理してくれる。頭の中でぐるぐるしていたものが文字になると、少し小さく見える。

---

## 6. 「休む判断」をAIに相談する

副業を休むかどうか、自分で判断すると罪悪感が出る。

だからAIに聞く。

```
今日の状態：[気分・体調]
副業の締め切りや急ぎの作業：[あり・なし]
休んでいいか判断してください。
```

「休んでいいです」と言われると、素直に休める。
自分に許可を出せない日に使う。

---

## 7. 「続けている事実」をAIに認めてもらう

週に1回、これを送る。

```
今週やったこと：[箇条書き]
「よく続けましたね」と言ってください。
```

バカみたいに見えるかもしれないけど、効く。

うつで無職で、それでも動いた。それを誰かに認めてもらうことが、次の週につながる。

---

## すぐ使えるプロンプトまとめ

**朝の気分チェックイン**
```
今日の気分：[0〜10]／体の状態：[  ]／今日できそうなこと：[  ]
今日1つだけタスクを決めてください。
```

**しんどい時の吐き出し**
```
今日しんどいです。作業の話じゃなくていいので、少し話を聞いてください。
```

**夜の振り返り**
```
今日やったこと（小さいことでも）：[  ]
「よくやりました」と言ってください。
```

**寝る前の不安整理**
```
明日が不安です。不安な理由：[  ]
最悪のケースと、実際にそうなる可能性を教えてください。
```

**休む判断**
```
今日の状態：[  ]／急ぎの作業：[あり・なし]
休んでいいか判断してください。
```

AIはその仕組みの一部になってくれる。完璧じゃないけど、一人よりずっと動きやすい。
