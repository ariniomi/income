# 無職で時間はある。でも動けない。その理由をAIに聞いたら、全部当たってた

**価格: 300円**
**対象: 時間があるのに動けない・副業を始めたいのに始められない人**

---

無職になって、時間だけはある。

1日24時間、全部自分のものだ。副業をやる時間なんて、いくらでもある。

なのに、動けない。

「なんで動けないんだろう」と思いながら、また一日が終わる。

ある日、その理由をChatGPTに聞いてみた。

「時間があるのに副業が始められません。なぜだと思いますか」

返ってきた答えが、全部当たっていた。

---

## AIが教えてくれた「動けない理由」5つ

---

### 理由1：選択肢が多すぎて、選べない（無料）

「何をすればいいかわからない」

これが一番多い。副業の情報は山ほどある。note、YouTube、ブログ、せどり、プログラミング——全部気になって、全部調べて、何も始まらない。

心理学では「決断疲労」と呼ぶらしい。選択肢が多いほど、人間の脳は疲弊して動けなくなる。

**AIの提案：「今日は選ばない。私が1つだけ決めます」**

「副業を1つだけ教えてください。選ばなくていいです」と送ると、AIが勝手に決めてくれる。選ぶコストがゼロになる。

---

### 理由2：「存在するだけ」でエネルギーを使い果たしている（無料）

うつの状態では、起きているだけで疲れる。

健康な人の「やる気がない」と、うつの人の「動けない」は別物だとAIは言った。健康な人は「エネルギーが余っているのに使いたくない」状態。うつの人は「そもそもエネルギーがほぼゼロ」の状態。

時間があっても、燃料がなければ動けない。

**AIの提案：「1分だけのタスクに分解する」**

「記事を書く」ではなく「タイトルを1つ考える」。「アカウントを作る」ではなく「登録ページを開く」。1分で終わることだけをタスクにする。

---

### 理由3：「やるべき」と「やりたい」の距離が遠すぎる（無料）

「副業しなきゃ」という気持ちはある。でも「副業したい」という気持ちはない。

義務感だけで動こうとすると、体が拒否する。これは意志の問題じゃなく、人間の構造の問題らしい。

**AIの提案：「『べき』を『してもいい』に変える」**

「今日note記事を書かなければならない」→「今日note記事を書いてもいい」

許可の形にするだけで、少し軽くなる。

### 理由4：失敗が怖くて、始められない

始めるということは、失敗する可能性が生まれるということだ。

動かなければ、失敗しない。だから動かない。

これは無意識に起きる自己防衛で、うつの人には特に強く出るとAIは言った。

**AIの提案：「失敗を『データ』と定義し直す」**

「この記事が売れなかった＝何がダメだったかのデータが取れた」

失敗ではなく実験。結果ではなく情報。この言い換えだけで、始めるハードルが下がる。

---

### 理由5：締め切りがないと、人間は動けない

会社員のとき、なぜ動けたか。締め切りがあったからだ。

無職になって締め切りが消えた。その瞬間に、「今日じゃなくていい」が無限に続くようになった。

**AIの提案：「AIに締め切りを作ってもらう」**

```
今日の副業タスクを1つ決めて、
「今日の23時までにやること」として宣言してください。
終わったら報告します。
```

AIが宣言してくれると、「誰かに言った」感覚が生まれる。それだけで動けることがある。

## 今日からすぐ使える「動けない日専用プロンプト」5つ

**① 選択肢を全部消す**
```
副業を始めたいです。
選択肢を考えたくないので、
私に一番合いそうなものを1つだけ決めてください。
条件：初期費用ゼロ、スマホでできる、1日30分以内
```

**② 1分タスクに分解する**
```
今日の副業タスク「[やること]」を、
1分以内に終わる作業に分解してください。
最初の一歩だけ教えてください。
```

**③ 「べき」を「してもいい」に変換する**
```
以下の文章の「べき」「しなければ」を
「してもいい」「できる」に書き換えてください：
[今日の自分の独り言を貼る]
```

**④ 今日の締め切りを作る**
```
今日中にやること：[タスク]
「今日の[  ]時までにやります」と
私の代わりに宣言してください。
```

**⑤ 動けなかった日を責めない**
```
今日も動けませんでした。
責めないで、明日の「最初の1分」だけ決めてください。
```
