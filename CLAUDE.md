# CLAUDE.md

このファイルは、Claude Code (claude.ai/code) がこのリポジトリで作業する際のガイダンスを提供します。

## プロジェクト概要

Note.com で有料記事を販売して月3,000円を目指すプロジェクト。

## ディレクトリ構成

- `articles/drafts/` — 執筆中の記事
- `articles/published/` — Note に公開済みの記事
- `research/` — トレンド調査・ネタ出し
- `tracking/` — 売上・PV の記録

## 記事執筆の方針

- 読者が「買ってよかった」と思える具体的な内容にする
- 1記事あたり100〜300円で販売
- 月3,000円 = 10〜30記事の売上が目標
